# Client

go to `"/Contact_List/"`

## Install dependencies (once)
```bash
npm install node
```
```bash
npm install vue
```

## Start Client
```bash
npm run dev
```

# Server

go to `"/server"`

## Install dependencies (once)
```bash
npm install express
```
```bash
npm install body-parser
```
```bash
npm install mongodb
```
```bash
npm install cors
```
```bash
npm install jsonwebtoken
```

## Start Server
```bash
node server.js
```

# Database
To initialize the database once, start the mongodb on `"localhost:27017"`

go to `"/server/init"`

## Init Database (once)
```bash
node initContactList.js
```
